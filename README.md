## Objetivo del proyecto
Analizar los datos del Titanic utilizando Pandas para mirar las variables más importantes, sacar estadísticas y entender quiénes tenían más probabilidades de sobrevivir.

##  Instrucciones para ejecutar el proyecto
1. Descargar los archivos `train.csv` y `test.csv` desde Kaggle:  
    [https://www.kaggle.com/competitions/titanic/data](https://www.kaggle.com/competitions/titanic/data)
2. Guardarlos dentro de la carpeta `data/`.
3. Abrir el notebook principal en `notebooks/`.
4. Ejecutar las celdas en orden para reproducir el análisis.
